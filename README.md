# 별마을

마인크래프트 Star Technology 모드팩 기반의 공장 & 경제 팀 스카이블록 서버를 소개하는 공식 블로그입니다.

## 소개

별마을은 Star Technology 모드팩을 중심으로 한 팀 기반의 공장 자동화 및 경제 중심 마인크래프트 스카이블록 서버를 소개하는 블로그 사이트입니다.

## 주요 시스템

- 공장 자동화 시스템
- 경제 중심 게임플레이
- 팀 기반 협력
- Star Technology 모드팩 활용
